FreeCAD MCP

FreeCAD MCP is an open-source Model Context Protocol (MCP) server that enables AI assistants to interact with FreeCAD, the popular open-source parametric 3D CAD modeler. By implementing the MCP standard, FreeCAD MCP allows AI tools like Claude, ChatGPT, and other LLMs to directly control and automate FreeCAD workflows.

What is FreeCAD MCP?

FreeCAD MCP bridges the gap between AI assistants and CAD automation. It provides a standardized interface for AI models to:

  • Create and modify 3D models programmatically
  • Extract design parameters and features
  • Generate technical drawings automatically
  • Perform batch operations on CAD files
  • Integrate CAD workflows with AI-powered design tools

Key Features

The FreeCAD MCP server implements the Model Context Protocol specification, providing:

  • Full FreeCAD API Access - Complete control over FreeCAD's Python API
  • Sketch Creation - Automated 2D sketch generation with constraints
  • 3D Operations - Extrude, revolve, loft, and other 3D operations
  • Parametric Design - Modify parameters and see real-time updates
  • File Management - Open, save, and export in multiple CAD formats
  • Assembly Tools - Create and manage complex assemblies

Installation

Getting started with FreeCAD MCP is simple:

# Clone the FreeCAD MCP repository
git clone https://github.com/contextform/freecad-mcp.git

# Install dependencies
cd freecad-mcp
npm install

# Start the MCP server
npm start

Use Cases

FreeCAD MCP enables powerful automation scenarios:

  • AI-Assisted Design - Let AI help you create complex geometries
  • Batch Processing - Automate repetitive CAD tasks
  • Design Optimization - Use AI to optimize parameters
  • Documentation Generation - Auto-generate technical drawings
  • Quality Checks - Automated design validation and testing

Why FreeCAD MCP?

Traditional CAD automation requires deep knowledge of scripting languages and CAD APIs. FreeCAD MCP changes this by allowing you to describe what you want in natural language, and letting AI handle the implementation details. Whether you're a mechanical engineer, product designer, or hobbyist, FreeCAD MCP makes CAD automation accessible to everyone.
